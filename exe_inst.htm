Hamster-Installation mittels EXE-Datei

Grundsätzlich sollte man zur Installation von Software keine EXE-Dateien aufrufen, da diese i.d.R. diverse Risiken bergen. Für den Hamster steht trotzdem eine solche Datei zur Verfügung. Hamsterclassic.exe wurde mit dem in Windows enthaltenem Programm iexpress.exe erstellt und führt das gleiche Script aus wie hier beschrieben. Auch diese Installation muß "als Administrator" ausgeführt werden und erfordert mindestens Windows 7.

Für maximale Kontrolle können Sie Hamsterclassic.exe aus dem CAB-Archiv selbst erzeugen. Hierzu laden Sie außer der CAB-Datei noch die Datei ham_create_sed.cmd herunter und führen Sie diese aus. Das Ergebnis der Konvertierung ist die Datei hamsterclassic.exe
