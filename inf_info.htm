Hamster-Installationsscript für den Windows-Installer (ab Windows 7)

Aufruf mit Rechtsklick/Installieren oder Kommandozeile:
%SystemRoot%\System32\InfDefaultinstall.exe "<path>\Hamster-Setup.inf"

Merkmale:
  • saubere, vollständige Hamster-Installation mit 3 Klicks möglich
  • Programm und Daten werden getrennt abgelegt, so daß ein sicherer Betrieb bei Rechtetrennung und aktivierten SRP möglich ist
  • die voreingestellten Verzeichnisse sind kompatibel mit den Windows-Richtlinien
  • bei Folge-Installationen/Updates werden die bisher verwendeten Verzeichnisse vorgeschlagen und Empfehlungen für den ggf. besseren Ort gegeben
  • Hamster wird bei der Installation als OLE-/COM-/DCOM-Server registriert
  • Hamster läßt sich aus den Windows-Einstellungen heraus komplett und rückstandsfrei deinstallieren
  • korrigierte deutsche Sprachdatei enthalten
  • Icons auf dem Desktop und in Autostart werden automatisch erstellt
  • OpenSSL wird in der neuesten Version von https://indy.fulgan.com/SSL und mit passenden Dateinamen direkt benutzbar eingerichtet
  • aktuelle Root-Zertifikate sind enthalten, gesicherte Verbindungen zu den meisten Servern sind somit möglich
  • in vielen Fällen keine Eingabe des Administrator-Kennworts notwendig
  • Installationsscript liegt im Klartext vor
  • Verwendung von Bordmitteln, keine suspekte .exe nötig
  • zusätzlich zur internen Hilfe ist eine PDF-Hilfedatei enthalten
  • das Script ist Open Source / Public Domain, kann also beliebig weitergegeben werden
Einschränkungen:
  • Auf 64-Bit-Systemen sollte das Script aus dem Windows-Datei-Explorer oder einem anderen 64-Bit-Programm ausgeführt werden.
  • Je nach Windows-Konfiguration müssen diverse Warnmeldungen bestätigt werden.
