Installationsarchiv für den Hamster Mail- und Newsserver.

Die Installation läuft wie folgt ab:
  • Laden Sie die Datei  Hamster-Setup.inf  herunter, Download-Link siehe unten.
  • Rufen Sie im Datei-Explorer (!) das Download-Verzeichnis auf.
  • Rechts-Klick auf „Hamster-Setup“.
  • Klick auf „Installieren“.
  • Bestätigen Sie ggf. Sicherheitsabfragen. Die nächste und letzte Abfrage ist die Sprach-Auswahl.
  • Außerdem sollten Sie ggf. die Unterstützung für HLP-Dateien (interne Hamster-Hilfe) mit WinHlpUpd.cmd aktivieren.
Die heruntergeladene Datei (Dateibeschreibung) muß zwingend die Endung INF tragen, was aus verschiedenen Gründen leider nicht zuverlässig funktioniert. Daher hier zwei Links für diese Datei:

Download-Link 1
Wenn die Datei als „.txt“ abgespeichert werden sollte, benennen Sie sie bitte um oder verwenden Sie diesen Link:

Download-Link 2
und rufen Sie in Ihrem Browser den Menüpunkt „Seite speichern unter“ (o.ä.) auf. Speichern Sie sie NICHT als Textdokument ab. Achtung, der Internet Explorer macht die Datei beim Abspeichern evtl. unbrauchbar. Firefox funktioniert.

Wenn die Datei als INF vorliegt und die Installation trotzdem nicht klappt, laden Sie von dieser Seite zusätzlich die CAB-Datei herunter. Führen Sie die obigen Klicks nun nochmals aus. Wenn auch das nicht funktioniert, entpacken Sie die cab-Datei vorher zusätzlich und führen das Script im Ziel-Ordner aus. (Sobald die Datei später mal auf dem „richtigen“ Webserver liegt, fallen diese Hürden weg.)

Der gesamte Download umfaßt weniger als 5MB.
